# UK Gadgets

Some small tools for Hongkongers that might be useful when studying/working in the UK.

Built using React.

## Installation

**USE node v14.9.0 (npm v6.14.17)**

`nvm use v14.9.0`

`npm install -g npm@6`

Requires Node.js and npm

```bash
npm install
```

or

```bash
yarn
```

## Usage

```bash
npm run start
```

or

```bash
yarn start
```
